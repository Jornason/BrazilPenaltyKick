巴西十二码
===

### 简介
1. 移动设备休闲小游戏，基于Cocos2d-x 3.0RC1 + SpriteBuilder 1.x + spritebuilder_cocos2dx/spritebuilder_1.1_beta1 开发的足球射门游戏
2. cocos2d-x 3.0文件需要另行下载，未包含在本项目文件列表

### 状态
开发中
